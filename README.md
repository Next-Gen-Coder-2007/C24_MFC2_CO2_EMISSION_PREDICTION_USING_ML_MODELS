## 📌 Project Overview

This project focuses on predicting CO₂ emissions using a combination of machine learning and deep learning models on a cleaned monthly sectoral dataset. The goal is to analyze historical emission patterns and build accurate forecasting models.

Multiple models such as LSTM, Ridge Regression, XGBoost, LightGBM, and Support Vector Machine (SVM) are implemented and compared. The dataset is preprocessed through missing value handling, normalization, and feature engineering using lag variables to capture time-based dependencies.

Each model is evaluated using standard performance metrics including RMSE, MAE, MSE, and R² Score. The project highlights the effectiveness of different approaches, where deep learning models like LSTM capture temporal trends, and ensemble methods like XGBoost and LightGBM provide strong predictive performance.

Overall, this project serves as a comparative study of various techniques for time-series forecasting of CO₂ emissions.
